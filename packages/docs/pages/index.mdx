## What is Kaito?

Kaito is a minimalistic, HTTP framework for Node.js and TypeScript. It can also work in JavaScript environments, but
this may change in the future.

<Callout emoji="‚ö†">Kaito is not production ready, please use it with caution</Callout>

### What is Kaito not?

Kaito is not:

- Stable
- Battle tested
- For writing really low amounts of code - for now. We'll eventually have a solid API for writing very little code
